# sahou (Rust runtime)

**Placeholder.** This crate reserves the `sahou` name on crates.io for the upcoming
Rust runtime — the app-facing library for sending and receiving typed messages over a
Sahou contract (the counterpart of `sahou` on PyPI and `sahou` on npm). It has no
functionality yet.

- Command-line tool: install `sahou-cli` (the `sahou` command).
- Schema core: `sahou-core`.

See the [project README](https://github.com/craf-te/sahou#readme).

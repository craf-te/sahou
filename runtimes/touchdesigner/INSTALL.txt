==============================================================================

REQUIREMENTS
  - macOS on Apple Silicon (arm64). Intel Macs are not supported.
  - TouchDesigner (Apple Silicon build).

INSTALL
  1. Unzip.
  2. Clear the download quarantine. This build is UNSIGNED, so without this step
     macOS refuses to load the plugins:
         xattr -cr SahouOut.plugin SahouIn.plugin
  3. Put the .plugin bundles where TouchDesigner scans for plugins — either:
       - next to your project:  <your .toe folder>/Plugins/
       - or user-global:        ~/Library/Application Support/Derivative/TouchDesigner099/Plugins/
     (copy them in, or symlink them.)
  4. Restart TouchDesigner (plugins are scanned at startup).
  5. Add a "Sahou Out" or "Sahou In" CHOP.

NETWORK PERMISSION (important)
  Sahou talks over your LAN via Zenoh. macOS gates local-network access per app,
  and the permission is granted to the HOST app (TouchDesigner), not to the plugin.
  Allow it under:
      System Settings -> Privacy & Security -> Local Network -> TouchDesigner
  Without this, discovery and receiving between apps/machines can silently fail.

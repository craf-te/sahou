# sahou (CLI)

The `sahou` command — a schema-first tool for building the interface layer between systems.
Describe messages and wiring in a schema; connect apps on a LAN without hand-managing IP
addresses or ports, with wrong types rejected at the boundary.

Install the prebuilt binary from the [GitHub Releases](https://github.com/craf-te/sahou/releases),
or from source with `cargo install sahou-cli` (installs the `sahou` command).

See the [project README](https://github.com/craf-te/sahou#readme) for full usage.

Apache-2.0.
